# 14 Precision Master (v4) Implementation Prompt

Author: Gemini CLI

## Prompt Purpose

This prompt specifies the **v4 Precision Master** extraction pipeline. It is the final "Production Grade" implementation, designed to maximize data density while maintaining perfect character accuracy and patient identity integrity.

Use this prompt to replicate the 1,032-cell, 50-row "Clean Diamond" Standard.

---

## Technical Architecture

### 1. Ultra-Dense Harvester (Stage 1)
- **Strategy:** Recursive Prose Mining + Semantic Sanitization.
- **Action:** 
    - Deduplicate merged table cells.
    - Extract all `Key: Value` pairs using regex split on clinical delimiters (`:`, `-`, `|`).
    - **Precision Guard:** Implement a `sanitize_value` function that trims clinical labels (e.g., "DOB", "NHS") from the end of values to prevent "label bleed" (e.g., converting `MALEDOB` to `Male`).
    - Capture source evidence snippets for every candidate.

### 2. High-Res Semantic Mapper (Stage 2)
- **Model:** `all-MiniLM-L6-v2` (Local).
- **Strategy:** Adaptive Matching.
- **Action:**
    - Map candidates to the 88 target columns using a **0.20 cosine similarity threshold**.
    - Implement "Winner-Takes-All": if multiple candidates match a column, keep the one with the highest confidence score.

### 3. Deterministic Identity Linker (Stage 4)
- **Strategy:** Sanitized Primary Key Grouping.
- **Action:**
    - Extract ONLY pure 10-digit integers for NHS numbers.
    - Group multiple documents by this clean NHS key (fallback to clean MRN).
    - **Identity Guard:** Ensure exactly 50 master records are generated by resolving all sub-document aliases.
    - **Additive Merge:** Consolidate journeys into one row, using "Sequential Drift" to fill follow-up columns (e.g., 2nd MRI) instead of overwriting baseline data.

### 4. Evidence Assembler (Stage 5)
- **Strategy:** Transparent Audit Trail.
- **Action:**
    - Assemble the final Excel using `openpyxl`.
    - Inject an **Excel Comment** into every non-empty cell containing the original source prose snippet.
    - Force **String-Encoding** on all cells to prevent scientific notation errors on identifiers.

---

## Validation Targets

- **Density:** > 1,000 cells.
- **Row Count:** Exactly 50.
- **Character Accuracy:** 100% precision on NHS/MRN/DOB (Zero label bleed).
- **Auditability:** 100% evidence coverage.

# 📊 Data Analysis

This folder contains analysis scripts, notebooks, and reports derived from the extracted clinical data.2

## 🎯 Objectives
1.  **Extraction Performance:** Evaluating the AI's accuracy against ground-truth datasets (e.g., F1-score per clinical field).
2.  **Clinical Outcomes:** Analyzing patient pathways, treatment wait times, and cohort staging distributions.
3.  **Data Quality:** Identifying missingness patterns in clinical MDT records.

## 📂 Structure
- `notebooks/`: Exploratory Data Analysis (EDA) in Jupyter format.
- `scripts/`: Reusable Python scripts for statistical analysis.
- `reports/`: Summary findings and visualizations (PDF/PNG).

## 🛠 Getting Started
To begin an analysis, load the extracted Excel files from the `baseline-solution/output/` or `data/` directories into a pandas DataFrame.
